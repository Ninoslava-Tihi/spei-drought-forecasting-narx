# DATA FOLDER

This folder contains the datasets used in this project.

## Description

The data stored in this folder are used for data preprocessing, model training, and model evaluation.

## Content

* Raw data files used as input for the analysis
* Processed datasets generated during preprocessing
* Additional data required for model validation or testing

## Notes

* Large datasets may not be included in the repository due to GitHub file size limitations.
* If the dataset is missing, please refer to the data source or instructions provided in the main project documentation.

## Usage

The scripts located in the `scripts` folder use the data from this directory for analysis, modeling, and prediction tasks.
